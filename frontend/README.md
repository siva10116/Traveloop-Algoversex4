# Traveloop

🚀 Traveloop Tech Stack (Hackathon)

## Frontend
• React + Vite
• Tailwind CSS

## Backend
• Node.js + Express.js

## Database & Auth
• Supabase (PostgreSQL + Auth + Storage)

## APIs
• GeoDB Cities API → City search
• Foursquare Places API → Activities & attractions
• Mapbox → Maps & itinerary visualization
• OpenWeather API → Weather forecast (optional)

## Charts
• Recharts / Chart.js

## Extra Features
• Public trip sharing
• Drag & drop itinerary
• AI trip planner using OpenRouter (optional)

## Deployment
• Vercel → Frontend
• Render / Railway → Backend

## Why this stack?
✅ Completely free/student-friendly
✅ Fast development
✅ Easy deployment
✅ Best for hackathon demos
✅ Scalable for future development
